# How To Set Up Enviorment and Run Application

### Install Flask (If not already installed)

Open Terminal <br>

`pip install flask`

### Run a Local Server

In Terminal type:<br>

`export FLASK_APP=recipeasy.app.py`

`flask run`

This should provide address on local machine:<br>
ex: * Running on http://127.0.0.1:5000<br>
